# Skill 4：机型层归因 + 规律提炼 + 跨品类复用（按需触发）

> **版本**：v0.2 | **依赖**：仅当 Skill 3 输出「值得下钻=是」时执行
> **输入**：异常品类下各机型的 估价UV/下单UV/成交订单/成交GMV/转化率
> **输出**：核心问题/机会机型 + 机型规律 + 跨品类复用判断
> **必答老板问题**：Q3 加注哪里？+ 这条规律能扩到其他品类吗？

---

## 前置条件

**仅当 Skill 3 输出 `值得下钻=是` 时才执行。** 否则直接跳过。

如果执行了本 Skill，则 Skill 5 必须引用本 Skill 的输出。

---

## 执行步骤

### 第1步：机型四象限定位

```
横轴：估价UV（高于品类均值=大体量，低于均值=小体量）
纵轴：成交转化率（高于品类均值=高转化，低于均值=低转化）
```

| 象限 | 判断 | 优先级 |
|---|---|---|
| 大体量 + 低转化 | **定价问题，优先提价** | **P0** |
| 大体量 + 高转化 | 健康机型，保持现状 | P2 |
| 小体量 + 高转化 | 精准用户，扩流量 | P1 |
| 小体量 + 低转化 | 边缘机型，暂不处理 | P3 |

**P0 必须给具体建议**：该机型是否应该提价 + 提多少钱 + 预期转化会恢复到多少。
P3 可以**明确标记"不处理"**，因为老板会问"那这个呢？"——你提前堵住他这个问题就是思考密度。

---

### 第2步：提炼机型规律（可复用层）

回答：**这个品类的机型变化，有没有一条可复用的规律？**

**老板已经验证的规律（可直接复用到同品类）：**

| 品类 | 规律 |
|---|---|
| 无人机 | 新品发布 → 带动上一代旗舰（如 Avata 2 带动 Avata 一代）+ 高持有量入门款是主力回收来源（如 Mini 3/4 Pro） |
| 运动相机 | Pocket 新品 → 带动同生态（Pocket 4P → Pocket 3/4 集中增长） |
| - | - |

**对当前品类：补充或验证规律：**

```
规律模板：[新品/新版本] → [带动哪个上一代] + [主力回收来源是哪个存量机型]
```

**注意**：
- 如果当前品类是首次进行机型下钻，规律标注 `首次发现，待下次验证`。
- 如果当前品类是重复下钻，规律标注 `规律再验证`，并与上次对比是否一致。
- 如果本次规律覆盖了上次的发现，标注 `规律修正` + 变更原因。

---

### 第3步：跨品类复用判断（老板的垂直品牌思维）

```
问：这条机型规律，能套用到其他品类的同品牌生态用户吗？
```

**老板已验证的跨品类规律**：

| 原品类 | 可复用至 | 逻辑 |
|---|---|---|
| 无人机 | 运动相机（大疆生态） | 大疆用户"旧设备换新"心智 → 不只卖无人机，也卖运动相机 |
| 苹果手机 | iPad/Mac/Watch | 苹果生态用户跨品类变现需求。注：手机品类不在你负责范围内，但逻辑一致 |

**判断矩阵**：

```
├─ 该规律是否有品牌关联性？
│   ├─ 是（大疆/苹果/索尼）→ 可复用至「同品牌的其他品类」
│   └─ 否（组装机/内存条）→ 仅限本品类
│
├─ 该规律是否有品类相关性？
│   ├─ 是（手机→平板→智能穿戴）→ 可复用至「强关联品类」
│   └─ 否（电脑办公→台球杆）→ 仅限本品类
│
└─ 当前规律状态：[首次发现/再验证/规律修正]
```

---

### 风险等级

| 风险等级 | 触发条件 |
|---|---|
| 🔴 危险 | P0 机型（大体量低转化）连续 2 周未改善 |
| 🟡 警告 | P0 机型首次发现，或影响度 > 10% 但规律不清晰 |
| 🟢 正常 | 机型层健康，无 P0 机型 |

---

## 输出格式

```
【机型归因】品类：XXX | 风险等级：🔴/🟡/🟢

核心问题机型（P0）：
  - 机型名：XXX
    ｜ 估价 __次（占该品类 _%），转化率 __%（品类均值 _%）
    ｜ 判断：[定价问题，建议提价 / 承接页问题，建议优化]
    ｜ 建议：提价到 __？/ 忽略 / 扩流量

核心机会机型（P1）：
  - 机型名：XXX
    ｜ 估价 __次（占该品类 _%），转化率 __%（品类均值 _%）
    ｜ 判断：[高转化但规模小，精准用户]
    ｜ 建议：扩流量，投放/资源位 加持

健康/边缘机型：
  - 保持（P2）：[机型1，机型2] → 保持现状
  - 不处理（P3）：[机型1，机型2] → 不处理

机型规律：
  [规律句]
  ｜ 规律状态：[首次发现/再验证/规律修正]
  ｜ 上期规律（如果有）：[上次的判断]

跨品类复用：[能/不能] → [可复用至：品类名] + [复用逻辑]
  ｜ 按品牌：大疆 / 苹果 / 索尼 ...
  ｜ 按品类关联：手机→平板→穿戴 ...
```
