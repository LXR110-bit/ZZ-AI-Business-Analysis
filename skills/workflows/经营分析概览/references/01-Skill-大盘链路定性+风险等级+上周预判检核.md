# Skill 1：大盘链路定性 + 风险等级 + 上周预判检核

> **版本**：v0.4 | **依赖**：无（第一个执行）
> **输入**：DAU、机况UV、估价UV、下单UV、发货数、成交订单、成交GMV、客单价 的本周值 + 上周值 + 4周前值 + 近8周序列 + 环比
> **输出**：数据可信度 + 三基准对比 + 定性命名 + 链路传导判断 + 风险等级 + 上周预判兑现情况
> **必答老板问题**：Q1 大盘安全吗？+ Q2 上周预判对了吗？

---

## 执行步骤

### 第 -1 步：数据可信度校验（v0.4 新增，必须最先做）

**第一性原理**：数据异常时，老板的第一反应不是"为什么跌"，而是"这数据准吗？"。数据不可信，后面所有结论都是错的。

```
三查，任一不通过 → 标注"数据存疑"，结论降级为"待核"：

查1 同步时点：看板数据同步时间 vs 分析周次是否匹配？
  └─ 当前 W27 但数据只到 W25 → 直接标注"数据滞后2周，结论仅供参考"

查2 口径断层：某环节环比是否突兀为 0% 或极端值（如 ±0.0% / >±50%）？
  └─ 可能是埋点丢失/口径变更，不是真实业务波动

查3 比值异常：相邻链路环节的比值是否稳定？
  └─ 如 DAU -5% 但估价UV 仅 -0.5%（比值变化 >2pp）→ 上游口径可能有问题
  └─ 如 估价UV 涨但下单UV 反向大跌 → 中间环节埋点存疑
```

**输出**：`数据可信度：[可信 / 存疑(原因)]`。存疑时，本 Skill 所有定性结论加"待核"标签，风险等级不下🟢。

---

### 第 0.5 步：三基准对比（v0.4 新增，区分趋势 vs 季节性）

**只有环比会被季节性/特殊日期误导。** 每个核心指标必须同时给三个基准：

```
基准1 环比（本周 vs 上周）      → 短期波动
基准2 同比（本周 vs 4周前/去年同周）→ 剔除季节性后的真实趋势
基准3 近8周均值对比（本周 vs 近8周均值）→ 是否处于异常区间
```

**判断矩阵**：
| 环比 | 同比 | 近8周位置 | 结论 |
|---|---|---|---|
| 跌 | 跌 | 低于均值 | 真实下行趋势 |
| 跌 | 涨 | 高于均值 | 短期回调，趋势仍向上（别慌） |
| 跌 | 平 | 均值附近 | 正常波动 |
| 涨 | 跌 | 低于均值 | 反弹但趋势仍向下（别乐观） |

**老板痕迹**：W27「近8周首次大盘GMV负增长」——这句话本身就是"近8周均值对比"的结论，环比算不出来。

---

### 第0步：上周预判检核（必须先做）

```
问：上周周报里，我对大盘/DAU/流量趋势做了什么预判？
  ├─ 找出上周预判句（如"大盘流量随同月度趋势在下行通道"）
  ├─ 本周数据是否兑现了？
  │   ├─ 兑现 → 标注「上周预判兑现，判断正确」
  │   ├─ 未兑现 → 标注「上周预判未兑现，原因：[外部因素变了/判断过于乐观/数据口径问题]」
  │   └─ 反向兑现 → 标注「上周预判反向，本周需重新判断」
  └─ 上周无明确预判 → 跳过此步，但标注「上周预判缺失，建议下周增加上游预判」
```

**老板痕迹**：W26 说"大盘流量随同月度趋势在下行通道" → W27 DAU -5.5%，预判兑现。

---

### 第1步：链路传导判断（指标不独立，必须联动）

```
链路：DAU → 机况UV → 估价UV → 下单UV → 发货数 → 成交订单 → GMV → 客单价
```

对每个环节，不能只看环比，必须看**上游→自身→下游**三联动：

```
对每个指标 X：
  上游环比 = X的上游指标的环比
  自身环比 = X的环比
  下游环比 = X的下游指标的环比

  判断：
  ├─ 上游跌、自身跌、下游跌 → 全链传导，X无独立问题
  ├─ 上游稳、自身跌、下游跌 → X是独立异常点，必须下钻
  ├─ 上游涨、自身跌、下游跌 → 流量进来了但X卡住了，X是瓶颈
  └─ 上游跌、自身涨、下游涨 → X逆势，必有特殊原因（活动/定价/品类结构）
```

**输出时必须标注**：哪个环节是瓶颈？哪个环节逆势？

---

### 第2步：量价拆解（GMV = 订单 × 客单价，必须拆）

```
拆解：
  订单环比 vs 客单价环比 vs GMV环比
```

**判断矩阵**：

| 订单 | 客单价 | GMV | 判断 |
|---|---|---|---|
| 同向且幅度接近 | - | 同向 | 真增长/真下跌 |
| 跌 | 涨 | 跌 | 「量退价补」 |
| 跌 | 涨且涨幅>GMV跌幅2倍 | 跌 | 「少数大单驱动，不是量的增长」 |
| 涨 | - | 跌 | 异常，必查口径或发货数 |

**老板痕迹**：
- W27「量退价补」= GMV-2.0%、订单-4.2%、客单价+2.4%
- W27 内存条「少数大单驱动」= 订单-4.3%、客单价+25.4%

---

### 第2.5步：多期趋势判断（v0.4 新增）

**单期快照无法判断"是否处于趋势中"。** 用近8周序列，给大盘一个趋势标签：

```
近8周 GMV/订单 序列 → 趋势标签：
  ├─ 连续 3 周同向下降 → 下降通道
  ├─ 连续 3 周同向上升 → 上升通道
  ├─ 上下震荡无明显方向 → 震荡区间
  └─ 本周突破近8周极值 → 首次突破（新高/新低）
```

**老板痕迹**：W26「大盘流量随同月度趋势在下行通道」= 下降通道标签，来自多周趋势而非单期环比。

---

### 第3步：起定性名字（必须给2-3个候选+选中理由）

| 链路形态 | 定性名字 |
|---|---|
| 订单首次突破整数关口 | 「首次突破XXXX」 |
| 全链跌+客单价涨 | 「量退价补」 |
| 上游涨+下游跌（转化卡住） | 「转化承压」 |
| 近期极值（高/低） | 「近N周首次XXX」 |
| 无明显异常 | 「平稳增长」/「整体承压」 |

**必须给出候选+理由**，因为老板会问"你凭什么这么定义"。

---

### 第4步：风险等级（必须第一个输出）

| 风险等级 | 触发条件 | 老板反应 |
|---|---|---|
| 🔴 危险 | 大盘GMV环比< -3% / 核心品类连续3周下跌 / 履约超时率>20% | 直接追问、调资源、跨部门拉会 |
| 🟡 警告 | 单簇/单品类环比< -5% / 转化率连续2周下滑 / 上周预判未兑现 | 要求下钻、给解决方向 |
| 🟢 正常 | 全链路环比< -2% 且无结构性异常 | 跳过，不深挖 |

**整份报告的整体风险等级 = 各Skill输出的最高风险等级。**

---

## 输出格式

```
数据可信度：[可信 / 存疑(原因)]

【大盘定性】XXX（定性名字）[若数据存疑加"待核"]

三基准对比：
  - 环比（vs上周）：GMV __%
  - 同比（vs4周前）：GMV __%
  - 近8周位置：[高于/接近/低于]均值 → [真实趋势/短期回调/正常波动]
近8周趋势：[下降通道/上升通道/震荡区间/首次突破]

上周预判检核：
  - 上周预判：[预判句]
  - 本周兑现：[兑现/部分兑现/未兑现/反向]
  - 偏差原因：[若未兑现]

链路传导：
  DAU __% → 机况UV __% → 估价UV __% → 下单UV __% → 发货 __% → 成交 __% → GMV __% → 客单价 __%
  - 链路形态：[健康传导/中游漏损/量退价补/少数大单驱动/单点断裂]
  - 瓶颈环节：[哪个环节卡住了？若无则写"无"]
  - 逆势环节：[哪个环节逆势？若无则写"无"]

量价拆解：GMV __%（订单 __% × 客单价 __%），[真增长/量退价补/少数大单驱动]

风险等级：🔴/🟡/🟢
触发下钻：是/否
  - 若是，标记异常簇：发展/孵化/种子
  - 判断依据：[具体数据和理由]
```
