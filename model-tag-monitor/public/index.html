观测的转化率(固定)

  • 估价完成率 = 估价UV / 机况UV
  • 估价下单率 = 下单UV / 估价UV
  • 估价发货率 = 发货量 / 估价UV
  • 估价成交率 = 成交量 / 估价UV
  • 质检退回率 = 退回量 / 质检量
