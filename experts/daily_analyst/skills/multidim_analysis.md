# 多维度数据分析 Skill

## 一、输入识别

提取：
- **指标**：要分析的目标指标（GMV / 转化率 / 客单价 等）
- **维度**：拆解的维度（品类 / 渠道 / 用户分层 / 时段 / 地域）
- **时间范围**：分析周期
- **对比对象**：vs 上期 / vs 去年同期 / vs 大盘

## 二、数据获取

1. `read_email` 取数据
2. `parse_csv` 解析
3. `split_dimension(data, by=<维度>)` 拆解
4. 对每个维度组合做 `calc_caliber` 同环比

## 三、分析路径

- §3 瓶颈定位：在所有维度组合里找绝对值最差的 / 变化最大的
- §1 三层穿透：每个异常维度都要回答三层
- §4 四问：每个变化 > 5% 的维度走四问
